# Project-1
# 개인 맞춤형 식단 관리 앱 (Personalized Diet Management App)

## 소개
이 앱은 사용자가 건강 목표에 맞춰 개인화된 식단을 계획하고 관리할 수 있는 웹 애플리케이션입니다. 음식 데이터베이스와 연동하여 영양소를 추적하고, 개인 맞춤형 식단을 추천합니다.

## 기술 스택
- **프론트엔드**: React
- **백엔드**: Flask (Python)
- **데이터베이스**: MySQL

## 설치 방법

### 백엔드 설치
1. `server/` 디렉토리로 이동합니다.
2. 필요한 패키지를 설치합니다:
   ```bash
   pip install -r requirements.txt
3. Flask 서버를 시작합니다
    python app.py
